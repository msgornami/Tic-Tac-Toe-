# **App Name**: BoltGrid

## Core Features:

- Seamless Telegram Auth: Enables instant access via Telegram Web App session and Firebase Anonymous authentication.
- Live Firestore Matchmaking: Uses Firestore to handle real-time game queues and instantly pair active players for a session.
- Reactive Multiplayer Grid: A high-performance game board that uses Firestore listeners to provide lag-free real-time state updates.
- Gen-AI Tactic Post-Mortem: Uses generative AI as a tool to analyze completed matches and provide players with personalized strategy improvements.
- Elite Hall of Fame: A high-performance leaderboard tracking and displaying the top 100 players globally based on competitive rank.
- Player Career Hub: Provides an overview of individual Win/Loss/Draw statistics and a complete history of past match performances.
- Instant Invite Links: Functionality to create and share private room links via the Telegram chat interface for playing with friends.

## Style Guidelines:

- Primary color: A vibrant Electric Indigo (#8476FF) to drive the high-stakes competitive atmosphere.
- Background: A deep, rich Midnight Ink (#17151F), ensuring a focused gaming environment while staying easy on the eyes.
- Accent: A brilliant Cosmic Blue (#4382FF) for secondary UI indicators, buttons, and victory animations.
- Headlines and interface controls use 'Space Grotesk' (sans-serif) for its high-tech, futuristic geometric qualities.
- Information blocks and statistics use 'Inter' (sans-serif) for its clinical clarity and objective look.
- Uses sleek, high-stroke-weight line icons that mirror the geometric lines of the Tic Tac Toe grid.
- Smooth spring-based micro-interactions for tile placement and layout shifts to enhance the tactile feel of the grid.
